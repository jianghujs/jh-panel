# 变更：在主从配置页添加半同步开关

## 原因
当前管理员需要在面板外手动配置 MySQL 半同步复制。他们希望在主从配置页内直接提供开关，自动写入必要配置、重启 MySQL，并根据主/从角色执行对应 SQL。

## 变更内容
- 在主从配置页新增 UI 控件，展示半同步启用状态并支持确认后启用或禁用。
- 新增后端接口，读取当前半同步状态，按需编辑 `my.cnf` 中的相关指令，并在状态变更时重启 MySQL。
- 当启用半同步后，在主模式或初始化从节点时执行 `rpl_semi_sync_*` SQL，确保运行状态与配置一致。
- 若写入配置或重启失败，需要保留错误信息并反馈给前端。

## 影响
- 影响的规范：replication-semi-sync（新增能力）
- 影响的代码：`index.html`（Tab 布局）、`js/mysql-apt.js`（masterOrSlaveConf 逻辑）、`index.py`（新接口、配置编辑、服务重启）
