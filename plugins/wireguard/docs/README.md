# WireGuard 插件使用与验证

## 手动验证清单
1. 进入“密钥初始化”，确认公钥/私钥为空时显示“生成密钥”。
2. 点击“生成密钥”，确认公钥/私钥内容展示，按钮变为“重新生成”。
3. 进入“配置列表”，点击“新增配置”，确认配置内容可编辑并保存为 `/etc/wireguard/<iface>.conf`。
4. 在“配置列表”点击“点对点配置向导”，填写对端信息并保存，确认配置文件自动创建。
5. 在“配置列表”执行“编辑/应用/删除”，确认配置可更新与生效。

## 注意事项
- 重新生成密钥会影响已存在的 WireGuard 配置，需确认后操作。
- 应用配置可能中断当前隧道连接，建议在维护窗口执行。
